# 部署到 GitHub Pages 并生成 APK 指南

## 快速步骤

### 第一步：部署到GitHub

1. 打开 https://github.com/new
2. 创建新仓库，命名为 `voice-translator`
3. 上传 `D:\voice-translator` 里的所有文件
4. 进入 **Settings → Pages**
5. Source 选择 **main branch**
6. 保存后会得到: `https://你的用户名.github.io/voice-translator`

### 第二步：生成APK

1. 打开 **https://www.appsgeyser.com/**
2. 选择 Website
3. 输入上面得到的 GitHub Pages 网址
4. 设置名称: "语音翻译器"
5. 点击 Generate
